# Rent-a-Skill

A Knowledge-as-a-Service portal for renting AI-powered skills, specialist workflows, and expert capabilities through a marketplace-style product experience.
[Live site](https://rent-a-skill.vercel.app)

## What This Project Covers

- AI skills marketplace prototyping
- Knowledge-as-a-Service positioning
- Reusable specialist workflow packaging

## Tech Stack

- Next.js
- TypeScript
- React
- Vercel
